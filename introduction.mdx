# Introduction to ValvePak

ValvePak is a .NET library for reading and extracting VPK (Valve Pak) files, the uncompressed archive format used to package game content in Source and Source 2 engine games.

## What is VPK?

VPK (Valve Pak) files are uncompressed archives that package game assets for Valve's Source and Source 2 engine games. These archives are used in popular games like Counter-Strike: Global Offensive, Dota 2, Team Fortress 2, and Half-Life: Alyx.

## Common use cases

- **Game modding**: Extract and modify game assets from Source engine games
- **Content analysis**: Inspect game files and their metadata
- **Asset extraction**: Extract textures, models, sounds, and other game resources
- **Custom packaging**: Create custom VPK archives for game mods or tools
