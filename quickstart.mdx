# Quick start

This guide will help you get started with ValvePak by walking through common operations.

## Read a VPK file

<Steps>
  <Step title="Create a Package instance">
    First, create a new `Package` instance. Always use `using` to ensure proper disposal:
    
    ```csharp
    using SteamDatabase.ValvePak;
    
    using var package = new Package();
    ```
  </Step>
  
  <Step title="Open the VPK file">
    Call the `Read` method with the path to your VPK file. For multi-chunk archives, use the `_dir.vpk` file:
    
    ```csharp
    // Open a vpk file
    package.Read("pak01_dir.vpk");
    
    // Can also pass in a stream
    package.Read(File.OpenRead("pak01_dir.vpk"));
    ```
    
    <Note>
    Files such as `pak01_001.vpk` are just data files. You must open the `pak01_dir.vpk` directory file.
    </Note>
  </Step>
  
  <Step title="Find and read a file">
    Use `FindEntry` to locate a file, then `ReadEntry` to extract its contents:
    
    ```csharp
    // Find a file, this returns a PackageEntry
    var file = package.FindEntry("path/to/file.txt");
    
    if (file != null) {
        // Read a file to a byte array
        package.ReadEntry(file, out byte[] fileContents);
        
        // Inspect entry metadata
        Console.WriteLine(file.GetFullPath());  // "path/to/file.txt"
        Console.WriteLine(file.GetFileName());  // "file.txt"
        Console.WriteLine(file.TotalLength);    // file size in bytes
        Console.WriteLine(file.CRC32);          // CRC32 checksum
    }
    ```
  </Step>
  
  <Step title="Verify integrity (optional)">
    Optionally verify the file integrity using checksums and signatures:
    
    ```csharp
    // Verify MD5 hashes and signatures of the file if there are any
    package.VerifyHashes();
    ```
  </Step>
</Steps>

## Complete example: Extract all files

Here's a complete example that extracts all files from a VPK archive:

```csharp
using SteamDatabase.ValvePak;

using var package = new Package();
package.Read("pak01_dir.vpk");

foreach (var group in package.Entries)
{
    foreach (var entry in group.Value)
    {
        var filePath = entry.GetFullPath();
        
        package.ReadEntry(entry, out byte[] data);
        
        // Create the directory if needed, then write the file
        Directory.CreateDirectory(Path.GetDirectoryName(filePath));
        File.WriteAllBytes(filePath, data);
    }
}
```

## Optimize for many lookups

If you need to look up many files, call `OptimizeEntriesForBinarySearch()` before `Read()` to enable binary search instead of linear scan:

```csharp
using var package = new Package();

// Call before Read() to enable binary search for FindEntry
package.OptimizeEntriesForBinarySearch();
package.Read("pak01_dir.vpk");

// FindEntry calls are now significantly faster
var file = package.FindEntry("path/to/file.txt");
```

<Info>
You can also pass `StringComparison.OrdinalIgnoreCase` to `OptimizeEntriesForBinarySearch()` for case-insensitive lookups.
</Info>

## Stream-based access

For large files, use `GetMemoryMappedStreamIfPossible` to avoid loading the entire file into memory:

```csharp
var entry = package.FindEntry("path/to/large_file.dat");

using var stream = package.GetMemoryMappedStreamIfPossible(entry);
// Use the stream to read the file incrementally
```

This method returns a memory-mapped stream for large files (over 4 KiB) and a `MemoryStream` for smaller ones.

## Create a VPK file

You can also create new VPK archives:

```csharp
using var package = new Package();

// Add files to the package
package.AddFile("path/to/file.txt", File.ReadAllBytes("file.txt"));
package.AddFile("models/example.vmdl", File.ReadAllBytes("example.vmdl"));

// Remove a file from the package
package.RemoveFile(package.FindEntry("path/to/file.txt"));

// Write the package to disk
package.Write("pak01_dir.vpk");
```
